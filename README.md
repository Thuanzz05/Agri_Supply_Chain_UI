# Agri_Supply_Chain_UI

Agricultural Supply Chain Management System built with React + TypeScript + Vite

## Getting Started

This project provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

### Development

```bash
npm install
npm run dev
```

### Build

```bash
npm run build
```

## Tech Stack

- React 18
- TypeScript
- Vite
- ESLint
